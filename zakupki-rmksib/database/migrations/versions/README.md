# Миграции базы данных

Этот каталог содержит миграции Alembic для управления схемой базы данных.

## Существующие миграции

- `001_initial_tables.py` - Создание базовых таблиц (lots, suppliers)
- `002_add_users_table.py` - Добавление таблицы пользователей (users)

## Как использовать

### Применить все миграции
```bash
alembic upgrade head
```

### Создать новую миграцию
```bash
alembic revision --autogenerate -m "описание изменений"
```

### Откатить последнюю миграцию
```bash
alembic downgrade -1
```

### Просмотреть текущую версию
```bash
alembic current
```

## Структура миграции

Каждая миграция содержит:
- `upgrade()` - функция для применения изменений
- `downgrade()` - функция для отката изменений
- `revision` - уникальный идентификатор миграции
- `down_revision` - ссылка на предыдущую миграцию
