
## Содержимое

### 1. Ноутбук для Google Colab

**Файл:** `Ноутбук/neuro_employee_colab_final.py`

Финальная версия ноутбука с:
- ✅ Friendly user interface (интерактивный ввод данных)
- ✅ Главная функция `main()` для запуска
- ✅ Все функции разбиты на отдельные модули
- ✅ Чистые функции (без глобальных переменных)
- ✅ Поддержка всех режимов работы:
  - Базовый анализ лота
  - Расширенный анализ с оценками
  - Поиск поставщиков
  - Демонстрация всех функций

**Инструкция по использованию:**
1. Откройте Google Colab: https://colab.research.google.com/
2. Создайте новый ноутбук
3. Скопируйте код из `neuro_employee_colab_final.py` в ячейки
4. Разбейте код по разделам (см. комментарии `# ЯЧЕЙКА`)
5. Запустите все ячейки последовательно
6. В последней ячейке запустите `await main()`

### 2. Автоматический тестировщик

**Файл:** `Тестировщик/experiment_runner.py`

Автоматический скрипт для запуска серии экспериментов с различными параметрами.

**Использование:**
```bash
# Запуск всех экспериментов
python experiment_runner.py --api-key YOUR_API_KEY

# Запуск конкретных экспериментов
python experiment_runner.py --api-key YOUR_API_KEY --experiments 1,2,3

# Указание файла для результатов
python experiment_runner.py --api-key YOUR_API_KEY --output results.json
```

**Эксперименты:**
1. Базовый промпт для анализа лота
2. Расширенный промпт с оценками
3. Базовый анализ с temperature=0.1
4. Базовый анализ с temperature=0.5
5. Расширенный анализ с temperature=0.1
6. Расширенный анализ с temperature=0.7
7. Поиск поставщиков (5 компаний)
8. Поиск поставщиков (10 компаний)
9. Расширенный анализ с sonar-pro
10. Базовый анализ с увеличенным max_tokens

### 3. Документ с экспериментами

**Файл:** `Эксперименты/Эксперименты_нейро-сотрудник.md`

Подробное описание всех экспериментов с:
- Промптами для каждого эксперимента
- Параметрами (модель, temperature, max_tokens)
- Результатами экспериментов
- Выводами и рекомендациями
- Итоговыми выводами по всем экспериментам

**Формат:** Markdown (можно конвертировать в Word/Excel)

### 4. Документация

**Файлы:**
- `NEURO_EMPLOYEE_REPORT.md` - Отчет по Этапу #3 в формате Markdown
- `NEURO_EMPLOYEE_REPORT.docx` - Отчет по Этапу #3 в формате Word
- `convert_to_word.py` - Скрипт для конвертации MD в Word

- Python 3.8+
- Библиотека `httpx`
- Perplexity API ключ

### Для тестировщика:
- Python 3.8+
- Библиотека `httpx`
- Perplexity API ключ

### Для конвертации в Word:
- Python 3.8+
- Библиотека `python-docx`

