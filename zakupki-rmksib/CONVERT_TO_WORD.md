# Инструкция по конвертации отчета в Word

## Способ 1: Использование Pandoc (рекомендуется)

### Установка Pandoc:
- Windows: https://pandoc.org/installing.html
- Или через Chocolatey: `choco install pandoc`

### Конвертация:
```bash
pandoc NEURO_EMPLOYEE_REPORT.md -o NEURO_EMPLOYEE_REPORT.docx
```

## Способ 2: Использование онлайн-конвертеров

1. Откройте файл `NEURO_EMPLOYEE_REPORT.md` в текстовом редакторе
2. Скопируйте содержимое
3. Используйте один из сервисов:
   - https://www.markdowntoword.com/
   - https://dillinger.io/ (экспорт в Word)
   - https://stackedit.io/ (экспорт в Word)

## Способ 3: Ручное форматирование в Word

1. Откройте файл `NEURO_EMPLOYEE_REPORT.md` в текстовом редакторе
2. Скопируйте содержимое
3. Вставьте в Microsoft Word
4. Примените форматирование:
   - Заголовки: стили Заголовок 1, Заголовок 2, Заголовок 3
   - Код: моноширинный шрифт (Courier New)
   - Списки: автоматические списки Word
   - Таблицы: создайте таблицы для схем

## Способ 4: Использование Python (pypandoc)

```python
import pypandoc

# Конвертация MD в DOCX
output = pypandoc.convert_file('NEURO_EMPLOYEE_REPORT.md', 'docx', 
                                outputfile="NEURO_EMPLOYEE_REPORT.docx")
```

Установка: `pip install pypandoc`

## Рекомендации по форматированию в Word

1. **Титульная страница**: Добавьте титульную страницу с названием проекта
2. **Оглавление**: Вставьте автоматическое оглавление (Ссылки → Оглавление)
3. **Схемы**: Создайте схемы в Word используя инструменты рисования или SmartArt
4. **Таблицы**: Используйте таблицы для структурированных данных
5. **Изображения**: Добавьте скриншоты работы ноутбука Colab
6. **Нумерация страниц**: Вставьте номера страниц (Вставка → Номер страницы)

## Структура схемы в Word

Для создания схемы нейро-сотрудника в Word:

1. Вставка → Фигуры → Блок-схема
2. Используйте прямоугольники для модулей
3. Используйте стрелки для связей
4. Добавьте текстовые блоки с описаниями

Или используйте SmartArt:
1. Вставка → SmartArt → Иерархия
2. Выберите подходящий шаблон
3. Заполните текстом из отчета
